# Changelog

## [0.1.0] — 2025-06-01

### Added
- Initial release of Zoya programming language
- Core interpreter (lexer, parser, AST, runtime)
- REPL with history and tab completion
- Standard library modules: math, string, random, time, file, json, network, audio, physics
- 2D game engine (pygame-ce integration)
- 3D engine (Ursina/Panda3D integration)
- AI integration (Gemini, OpenAI, Ollama, LM Studio)
- CLI with file execution, one-liner, and examples
- Comprehensive test suite
