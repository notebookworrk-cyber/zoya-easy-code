# Zoya

A pure-Python SDK with 15 modules and zero external dependencies.

- **AI** — LLM providers, agents with ReAct loops, tool registries, RAG, embeddings
- **Web** — FastAPI-like routing, middleware, WebSocket, templates
- **Cloud** — Auth, document DB, storage, realtime pub/sub, multiplayer
- **Data Science** — DataFrame, 10+ ML models (RandomForest, SVM, KMeans, PCA), statistics
- **Mobile** — UIKit-like framework with navigation
- **Security** — Encryption, hashing, validation, sanitizer
- **DevOps** — CI/CD pipeline builder, container manager
- **Enterprise** — RBAC, SSO, audit logging, feature flags, rate limiter
- **Robotics** — Drone control, simulation, sensor fusion
- **Scientific** — Linear algebra (SVD, QR, eigendecomposition), PDE solvers
- **Desktop** — Cross-platform GUI framework
- **+4 more modules**

## Quick Start

```bash
pip install zoya-lang
```
