# Style Guide

## Python
- Follow PEP 8
- Use type hints (PEP 484)
- Line length: 100 characters
- Quotes: double quotes
- Indentation: 4 spaces

## Testing
- Use pytest
- Test files: `tests/test_*.py`
- Minimum coverage: 10%

## Linting
- Run: `ruff check .`
- Format: `ruff format .`

## Commits
- Follow Conventional Commits
- Prefix: feat, fix, docs, style, refactor, test, chore
